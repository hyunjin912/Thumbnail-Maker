# Thumbnail-Maker App (React ver.)

Unsplash API를 활용해 이미지 검색 애플리케이션을 만들었습니다.  
또한, html2canvas lib를 통해 커스텀된 썸네일을 다운 받을 수 있습니다.  
  
[DEMO](https://hyunjin912.github.io/p_thumbnail-maker)
